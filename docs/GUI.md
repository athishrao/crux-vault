# Simple GUI

## Streamlit Viz

Crux Vault features a user-friendly graphical interface built with Streamlit, making it easy to interact with the app on a browser.

The GUI allows you to:

- Easily navigate through vault features.
- Add, view, and manage your entries with a clean interface.
- Search and filter stored items efficiently.
- Import or export secrets right in the browser
- Securely interact with your vault data in a visual environment.

## Running the GUI

#### Launch the app:

`streamlit run frontend/simple_ui.py`

Streamlit usually opens the app on port 8050.

It also prints the local URL on start up. Open the provided local URL in your browser to access the GUI.
